# inboxproof

## API: Cloudflare DNS-over-HTTPS JSON API (primary) + Google Public DNS DoH JSON (fallback). 

Both providers are in the api-evangelist-network index, see [here](https://github.com/api-evangelist/api-evangelist-network).

## Pitch

Paste a domain, get an instant graded report card showing whether its email will land in spam — SPF, DKIM, DMARC, MX, BIMI — with a shareable, white-labelable PDF.
